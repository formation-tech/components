# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-06-13

### Added

- `<ft-owl-logo>` component: formation.tech owl logo with blinking eyes
- Mouse tracking for the eyes
- Eye rotation animation via the `loading` attribute
- Web component exports: `@formation.tech/components` and `@formation.tech/components/ft-owl-logo`
- React exports via `@lit/react`: `@formation.tech/components/react` and `@formation.tech/components/react/ft-owl-logo`
- Configurable props: `width`, `height`, `label`, `loading`
- `prefers-reduced-motion` support for blinking
- Local demo in `demo/`
